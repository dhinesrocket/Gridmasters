STYLING THEME:
===============

Colors:
  Primary: #00ff00 (bright green)
  Dark BG: #0a0e14 (terminal background)
  Accents: rgba(0, 255, 0, 0.X) (transparent green)

Effects:
  Glow: box-shadow: 0 0 Xpx rgba(0, 255, 0, 0.X)
  Text Shadow: 0 0 Xpx rgba(0, 255, 0, 0.X)
  Transitions: 0.3s - 0.8s ease-out

Animations:
  - Water draining (height transition)
  - Wave movement (translateX/Y)
  - Bottle drop (scale + translateY)
  - Pulse effect (opacity + scale)
  - Hover effects (transform scale)

Typography:
  Font: IBM Plex Mono (monospace)
  Sizes: 9px - 16px
  Weights: 400 (normal), 600 (semibold)
  Letter-spacing: 0.5px - 2px
